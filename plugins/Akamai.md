Akamai
========
Purges URLS from the Akamai CDN using the Content Control Utility API

Hooks
-----
- `cdn_purge`

Configuration
-------------
```yaml
plugins:
  akamai:
    username: foo
    password: bar
```

#### username
This is the username of the Akamai account you want to use

- Type: `String`

#### password
This is the username of the Akamai account you want to use

- Type: `String`
